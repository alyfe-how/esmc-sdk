# Contributing to ESMC SDK

Thank you for your interest in ESMC SDK!

## Important Notice

ESMC SDK is **proprietary software**. The source code in this repository is obfuscated and protected. We do not accept code contributions to the core SDK.

## How You Can Help

### Report Bugs

Found a bug? Please report it:

1. **Discord:** [discord.gg/N4qNFYWRwt](https://discord.gg/N4qNFYWRwt) (fastest response)
2. **GitHub Issues:** Open an issue with:
   - ESMC version (`esmc status`)
   - Node.js version (`node --version`)
   - Operating system
   - Steps to reproduce
   - Expected vs actual behavior

### Request Features

Have a feature idea? We'd love to hear it:

1. Join our [Discord](https://discord.gg/N4qNFYWRwt)
2. Share your idea in #feature-requests
3. Vote on existing feature requests

### Documentation

Help improve documentation:

- Fix typos or unclear instructions
- Add examples for your use case
- Translate documentation

Submit documentation improvements via GitHub Issues or Discord.

### Share Your Experience

- Write blog posts about ESMC
- Create tutorial videos
- Share on social media

## Code of Conduct

- Be respectful and inclusive
- No spam or self-promotion
- Help others when you can
- Follow Discord server rules

## Questions?

- **Discord:** [discord.gg/N4qNFYWRwt](https://discord.gg/N4qNFYWRwt)
- **Email:** support@esmc-sdk.com
